Modifier les styles d'un élément HTML

On utilise l'attribut element.style suivi de l'attribut CSS qu'on veut modifier.
Exemple : element.style.backgroundColor = "#FF0.
Remarquez que pour les attributs CSS composés comme border-left-color, on enlève le tiret et on met le caractère suivant en majuscule : borderLeftColor
